# Monthly Budget Excel

This project is a personal monthly budget tracker built in Excel to help organize income, expenses, and financial planning.

## 📊 Overview

The spreadsheet allows users to track their monthly finances in a simple and structured way, providing clear visibility into spending habits and financial balance.

## ⚙️ Features

- Income and expense tracking
- Categorization of expenses
- Automatic balance calculation
- Visual highlights for overspending
- Monthly financial summary
- Simple and user-friendly layout

## 📈 Insights

This project demonstrates:
- Data organization and structuring
- Use of Excel formulas for automation
- Basic financial analysis
- Creation of a simple data-driven solution for real-life problems

## 🛠️ Tools Used

- Microsoft Excel
- Formulas (SUM, IF, etc.)
- Conditional formatting
